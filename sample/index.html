このページは、GitHub Pages を利用した静的ホスティングの確認用ページです。

見出し（大） - GitHub Pagesとは？

GitHub Pagesは、HTMLやCSSなどの静的ファイルをGitHubリポジトリから直接Web公開できるサービスです。

見出し（中） - どんな用途に使える？

ポートフォリオ、プロジェクト紹介、ドキュメントの公開などに適しています。

見出し（小） - 特徴

無料・HTTPS対応・独自ドメインの設定も可能です。
